Introduction¶

In this project we take a look at TMDb which contains data regarding nearly 10,000 movies from 1960's up until 2010's.¶

This dataset contains alot of useful information including but not limited to genre, user ratings and revenue, and by analzing all of this information we are going to able to find relationships between them and aquire an an insight for what dectates a successful movie.¶

We will mainly look into these following questions:¶

  • How did different genres develop over the years regarding popularity?
  • What kinds of properties are associated with movies that have high revenues?

## Define function to expand columns that contain values separated by '|' such as 'genre' to correctly analyze each value

def tidy_split(df, column, sep='|', keep=False):
    """
    Split the values of a column and expand so the new DataFrame has one split
    value per row. Filters rows where the column is missing.

    Params
    ------
    df : pandas.DataFrame
        dataframe with the column to split and expand
    column : str
        the column to split and expand
    sep : str
        the string used to split the column's values
    keep : bool
        whether to retain the presplit value as it's own row

    Returns
    -------
    pandas.DataFrame
        Returns a dataframe with the same columns as `df`.
    """
    indexes = list()
    new_values = list()
    df = df.dropna(subset=[column])
    for i, presplit in enumerate(df[column].astype(str)):
        values = presplit.split(sep)
        if keep and len(values) > 1:
            indexes.append(i)
            new_values.append(presplit)
        for value in values:
            indexes.append(i)
            new_values.append(value)
    new_df = df.iloc[indexes, :].copy()
    new_df[column] = new_values
    return new_df

In the following analyses, we are going to mainly use the sum of values instead of the average, as some values only appear very few times and by using the average we would account for incorrect weight of these values.¶

Conclusions¶

There are independent characteristics that determine if a movie returns high revenue or gains high popularity.¶

Genres also play a role in that, as there are more dominant genres than other.¶

Top directors, actors and production companies dectates to a certain extent whether a movie returns high revenue.¶

Budget can dectate that as well, although all of the discussed variables do not correlate to ratings of a certain movie.¶

Limitations¶

The provided dataset includes a good portion of incorrect values such as 0 runtime, budget and revenue.Although 0 runtime values are clearly incorrect, but it is hard to tell if 0 budget and revenue values are, and there is no enough information to support either claims.¶

There is no proof that each release for each director, production comapny and cast member is listed in this dataset, and if that is the case, it may mean that results obtained here are inconclusive and may be skewed in favor to individuals or organisations listed abundantly in the dataset.¶

There is no clear definition for "Popularity" meter which may affect how we adress different questions such as the first question addressed in this report.¶

It is not made clear in the dataset if a specific director or cast member for example is playing a primary or secondary or even a cameo role in a given release, which may incorrectly favor results towards this individual. For example, a cast member having a cameo role in a given release may not have an impact on the return of that release.¶
